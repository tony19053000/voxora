# Voxora Privacy Policy

## Overview

Voxora is configured as a local-first desktop app. Meeting recordings, transcripts, summaries, and related metadata are intended to stay on the user's machine unless the user explicitly configures an external model provider.

## Local Data Handling

- Audio recordings are stored locally.
- Meeting transcripts and summaries are stored in the local application database.
- Exported files are written to user-selected local paths.
- Users remain responsible for securing their own device, filesystem, backups, and any external services they configure.

## External Providers

If you connect an external AI provider for summaries or transcription, data sent to that provider is governed by that provider's terms and privacy practices. Review those provider policies before enabling them.

## Analytics

This adapted Voxora build ships with analytics disabled. No outbound analytics initialization or tracking is intended to occur in normal operation.

## Updates

This adapted Voxora build ships with automatic update checks disabled.
